# Privacy Policy — VFA Documentation Standards

**Last updated: July 2026**

## What this extension does

VFA Documentation Standards is a Chrome browser extension that displays
organization documentation standards alongside EMSCharts PCR fields. It
operates only on `https://*.emscharts.com/pr/*` pages.

## Data collection

This extension **does not collect, store, transmit, or share any data** of
any kind, including:

- No personal information
- No patient health information (PHI)
- No usage or analytics data
- No crash reports or diagnostics

## Permissions used and why

| Permission | Reason |
|---|---|
| `activeTab` | Reads the active tab URL when the toolbar icon is clicked, so the PDF can open at the section matching the current EMSCharts page. No tab data is stored or transmitted. |
| Host permission: `https://*.emscharts.com/pr/*` | Required to inject the documentation button and field hints into EMSCharts PCR pages. The extension reads page structure only to find field elements; it does not read, capture, or transmit any field values or patient data. |

## Third-party services

This extension does not communicate with any external server or third-party
service. All content (the documentation standards PDF and scripts) is bundled
inside the extension and runs entirely on-device.

## Contact

For questions about this extension, contact the VFA organization administrator.
